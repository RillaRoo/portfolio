Roles
  • Game Designer
  • Sound Designer
    • Engine Unity


    "Cabrita Caganita" is a mobile platformer game that utilizes gyroscope mechanics to control the player.


    The game was created quickly as a way to explore the tilt control of the mobile device, and for that, a more humorous approach was chosen, where a goat defecates a ball (which is what the player controls). This ball can be alternated between a more rocky state and a more gelatinous state. If the player shakes the device, it executes the corresponding ability to the ball's state, where in the first one it performs a dash that breaks through walls and in the second one it can jump.


    The objective of the game is to complete the entire course in the shortest time possible.
